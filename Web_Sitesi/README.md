# ParametriX Web Sitesi

Next.js 14 (App Router) + TypeScript ile geliştirilmiş premium kurumsal web sitesi.

## Kurulum

```bash
cd Web_Sitesi
npm install
```

## Geliştirme

```bash
npm run dev
```

Site http://localhost:3000 adresinde çalışacaktır.

## Özellikler

- ✅ Next.js 14 App Router
- ✅ TypeScript
- ✅ Tailwind CSS
- ✅ Framer Motion animasyonları
- ✅ Responsive tasarım
- ✅ Portal Merkezi entegrasyonu
- ✅ Müşteri ve Yönetim panelleri
- ✅ Dark mode desteği
- ✅ Cookie consent
- ✅ Loading progress bar
- ✅ Back to top button

## Portal Sayfaları

- `/portal-selector` - Portal seçim sayfası
- `/customer-portal` - Müşteri portalı (Demo: demo@techvision.com / demo123)
- `/management` - Yönetim paneli (Demo: admin@rcsteknoloji.com / admin123)
